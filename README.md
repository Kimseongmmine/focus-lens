# FocusLens

웹캠을 활용해 사용자의 공부 상태를 기록하고,
집중 시간과 자리 비움 패턴을 분석하는 서비스입니다.

## 프로젝트 목표

- Java와 Spring Boot 백엔드 학습
- React 프론트엔드 학습
- 데이터베이스 설계 경험
- 소프트웨어 아키텍처와 책임 분리 학습
- GitHub 기반 개발 과정 기록
- 서비스 배포 경험

## 1차 MVP

- 과목 등록 및 조회
- 공부 세션 시작 및 종료
- 웹캠 화면 표시
- 사람 있음·없음 판단
- 자리 비움 시간 계산
- 공부 결과 조회

## 기술 스택

### Backend
- Java 21
- Spring Boot
- Spring Data JPA
- H2 Database

### Frontend
- React
- Vite

## 핵심 원칙

웹캠 영상은 서버에 전송하거나 저장하지 않습니다.

브라우저에서 영상을 분석한 뒤
`PRESENT`, `AWAY`와 같은 상태 정보만 서버에 전송합니다.
